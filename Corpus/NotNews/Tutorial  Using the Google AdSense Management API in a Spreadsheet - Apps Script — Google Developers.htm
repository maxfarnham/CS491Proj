    Tutorial: Using the Google AdSense Management API in a Spreadsheet

    Silvano Luciani, AdSense API Team
    December 2011

    Goal

    This tutorial shows how to integrate Google Sheets with the AdSense Management API to generate customized reports on the performances of an AdSense account.

    Time to Complete

    Approximately 30 minutes.

    Prerequisites

    Before you begin this tutorial, you should already have a Google AdSense account and be familiar with the AdSense inventory and reporting terminology. For section 5 you will need to have a Google Sites site.

    Overview

    You can use the AdSense Management API from Apps Script to do things like:

    • Generate a list of all Ad clients
    • Generate a list of all the Ad Units, custom channels and URL channels for a particular AdSense client ID
    • Generate reports based on many different dimensions and metrics, filtering the report by one or more of the inventory items: ad clients, ad units, custom channels and URL channels

    A full list of operations can be found in the reference documentation.


    In this tutorial we are going to see how to:

    • Section 1: Getting an API Key
    • Section 2: Get the ad clients for an AdSense account.
    • Section 3: Retrieve a report for an ad client account.
    • Section 4: Generate a Chart for your report data.
    • Section 5: Embed your Chart into a Google Sites page.
    • Summary

    In the first section we will see how to get an API key. In the second section we will see how to retrieve the list of ad clients id for an account. In the third section we will see how to use one of the retrieved ad clients id to generate a report on the performances of a specific ad client. In the fourth section we will see how to generate a Chart with the data retrieved from the API. Finally, in the fifth section we will see how to embed that Chart into a Google Sites page, by embedding the Google Apps Script in the page.


    Section 1: Getting an API key

    1. If you already have an API key, jump to Section 2.
    2. Go to the Google Developers Console.
    3. Register a project for your application, or use an existing one.
    4. You will find your API key in the API Access pane.
    5. To learn more about the Google Developers Console, please check the Google Developers Console Help.

    Section 2: Get the ad clients for an AdSense account

    1. Create a new spreadsheet.
    2. Add a sheet titled AdClients to your spreadsheet.
    3. Write in the cell A1 the value ID.
    4. Write in the cell B1 the value Product Code.
    5. Click Tools > Script Editor... to open the script editor.
    6. Click Resources > Advanced Google services....
    7. Enable the AdSense Management API and insert your API Key, then click OK.
    8. Give a name to the script and click OK.
    9. Copy and paste the following code into the editor.

      10. function adClients() {
              var ss = SpreadsheetApp.getActiveSpreadsheet();
              var sheet = ss.getSheetByName('AdClients');
              var clients = AdSense.Adclients.list().getItems();
              for (var i=0; i<clients.length; i++) {
          sheet      .getRange('A' + String(i+2)).setValue(clients[i].getId());
          sheet      .getRange('B' + String(i+2)).setValue(clients[i].getProductCode());
              }
      }

    10. Click the Save icon.
    11. Click the Run icon.
    12. The A column of the sheet AdClients has been populated with the list of IDs and the B column with the list of product codes for your AdSense account.

    Section 3: Retrieve a report for an ad client

    Now that we have the list of ad clients id for our AdSense account, we can choose one and retrieve a performance report for that specific ad client.


    1. Add a sheet titled Report to your spreadsheet.
    2. Write in the cells A1, B1, C1, D1 and E1 the following values:
      • Month
      • Page Views
      • Ad Requests
      • Matched Ad Requests
      • Individual Ad Impressions
    3. Click Tools > Script Editor... to open the script editor.
    4. Copy and paste the following code into the editor.

      5. function generateReport() {
              var ss = SpreadsheetApp.getActiveSpreadsheet();
              var sheet = ss.getSheetByName('Reports');
              var startDate = Browser.inputBox("Enter a start date (format: 'yyyy-mm-dd')");
              var endDate = Browser.inputBox("Enter an end date (format: 'yyyy-mm-dd')");
              var adClientId = Browser.inputBox("Enter an ad client id");
              var args = {
                'filter': ['AD_CLIENT_ID==' + adClientId],
                'metric': ['PAGE_VIEWS', 'AD_REQUESTS', 'MATCHED_AD_REQUESTS',
                           'INDIVIDUAL_AD_IMPRESSIONS'],
                'dimension': ['MONTH']};
              var report = AdSense.Reports.generate(startDate, endDate, args).getRows();
              for (var i=0; i<report.length; i++) {
                var row = report[i];
          sheet      .getRange('A' + String(i+2)).setValue(row[0]);
          sheet      .getRange('B' + String(i+2)).setValue(row[1]);
          sheet      .getRange('C' + String(i+2)).setValue(row[2]);
          sheet      .getRange('D' + String(i+2)).setValue(row[3]);
          sheet      .getRange('E' + String(i+2)).setValue(row[4]);
              }
      }

      Click the Save icon. From the AdClients sheet, select and copy the desired ad client ID. Click the Run icon. Insert a start date for our report, in the format yyyy-mm-dd. Insert an end date for our report, in the format yyyy-mm-dd. Paste the ad client id copied at 7 into the dialog box. The sheet Report now contains the number of page views, ad requests, matched ad requests and individual ad impressions for the selected ad client, broken down by month.

    Section 4: Generate a Chart for your report data


      Click Tools > Script Editor... to open the script editor. Copy and paste the following code into the editor. 
      function generateLineChart() {
              var doc = SpreadsheetApp.getActiveSpreadsheet();
              var startDate = Browser.inputBox("Enter a start date (format: 'yyyy-mm-dd')");
              var endDate = Browser.inputBox("Enter an end date (format: 'yyyy-mm-dd')");
              var adClientId = Browser.inputBox("Enter an ad client id");
              var args = {
                'filter': ['AD_CLIENT_ID==' + adClientId],
                'metric': ['PAGE_VIEWS', 'AD_REQUESTS', 'MATCHED_AD_REQUESTS',
                           'INDIVIDUAL_AD_IMPRESSIONS'],
                'dimension': ['MONTH']};
              var report = AdSense.Reports.generate(startDate, endDate, args).getRows();
              var tableBuilder = Charts.newDataTable()
                  .addColumn(Charts.ColumnType.STRING, "Month")
                  .addColumn(Charts.ColumnType.NUMBER, "Page views")
                  .addColumn(Charts.ColumnType.NUMBER, "Ad requests")
                  .addColumn(Charts.ColumnType.NUMBER, "Matched ad requests")
                  .addColumn(Charts.ColumnType.NUMBER, "Individual ad impressions");
              for (var i=0; i<report.length; i++) {
                var row = report[i];
          tableBuilder      .addRow([row[0],parseInt(row[1]),parseInt(row[2]),
              parseInt      (row[3]),parseInt(row[4])]);
              }
              var data = tableBuilder.build();
              var chart = Charts.newLineChart()
                  .setDataTable(data)
                  .setTitle("Performances per Month")
                  .build();

              var app = UiApp.createApplication().setTitle("Performances");
              var panel = app.createVerticalPanel();
        panel      .add(chart);
        app      .add(panel);
        doc      .show(app);
      }

    1. Click the Save
    2. From the AdClients sheet, select and copy the desired ad client ID.
    3. Click the Run
    4. Insert a start date for our report, in the format yyyy-mm-dd.
    5. Insert an end date for our report, in the format yyyy-mm-dd.
    6. Paste the ad client id copied at 2 into the dialog box.
    7. The active spreadsheet will now display a UI panel containing the chart. Something similar to the following picture:

    Section 5: Embed your Chart into a Google Sites page


    Step 1: Create the Google Apps Script


    1. Open your Google Site in the browser and log in with your administrative account.
    2. Click More > Manage site:

    3. In the left-hand sidebar, click Apps Scripts.
    4. Click Add new script.


    5. Copy and paste the following code into the editor, ensuring that you paste over the default function that is in the editor. Also, replace the placeholder values for ad client ID, start date and end date with appriopriate values:

      6. var app = UiApp.createApplication().setTitle("Performances");
              var adClient = 'INSERT_YOUR_AD_CLIENT_HERE';
              var startDate = 'INSERT_YOUR_START_DATE';
              var endDate = 'INSERT_YOUR_END_DATE_HERE';
              var args = {
                'filter': ['AD_CLIENT_ID==' + adClientId],
                'metric': ['PAGE_VIEWS', 'AD_REQUESTS', 'MATCHED_AD_REQUESTS',
                           'INDIVIDUAL_AD_IMPRESSIONS'],
                'dimension': ['MONTH']};
              var report = AdSense.Reports.generate(startDate, endDate, args).getRows();
              var data = Charts.newDataTable()
                  .addColumn(Charts.ColumnType.STRING, "Month")
                  .addColumn(Charts.ColumnType.NUMBER, "Page views")
                  .addColumn(Charts.ColumnType.NUMBER, "Ad requests")

                  .addColumn(Charts.ColumnType.NUMBER, "Individual ad impressions");

              }

                  .setTitle("Performances per Month")
                  .build();

              var app = UiApp.createApplication().setTitle("Performances");
              var panel = app.createVerticalPanel();
        panel      .add(chart);
        app      .add(panel);
              return app;

    6. Click the Save icon.
    7. Give the script a name and click Save.
    8. Close the Script Editor window.
    9. Click the < YOUR_SITE_NAME link. You have now created a script in Google Sites:

    Step 2: Create a Google Sites page to host the script


    1. On the top right hand corner of the Site, click the New Page icon.
    2. Choose the default Web Page template.
    3. In the Name field, type Chart Page.
    4. Click Create Page.

    Step 3: Embed the Google App Script in the Google Sites page


    1. Click Insert > Apps Script Gadget.
    2. Select the script created during Step 1. Upon selection, you will see a warning page which tells you that any user who is accessing the Sites page will run the script as YOU. This is a very important security consideration when you are developing scripts.
    3. Click Save.
    4. The final result will look like the following picture:

    Summary

    Congratulations! You have now learnt what you can do accessing the AdSense Management API from Google Apps Script.
